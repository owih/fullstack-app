# fullstack-app
<h1>Server side init</h1>

## database
Create database and write this one data to .env file

## Go to server folder
```
cd server
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

## Go to client folder
```
cd client
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
